# MetaPlay

**Metas grandes, movimientos pequeños, acción constante.**

MetaPlay es una Progressive Web App (PWA) diseñada para iOS que te ayuda a definir metas, dividirlas en acciones concretas y avanzar de forma constante sin presión.

## Características

- **Modo enfoque** — Te sugiere la siguiente acción más importante para que siempre sepas qué hacer
- **Sistema de niveles** — Gana XP al completar tareas y sube de Principiante a Leyenda
- **Racha de días** — Registra tus días consecutivos de actividad
- **Prioridades y energía** — Clasifica metas por impacto (alto/medio/ligero) y tipo de energía (foco profundo/rápida)
- **Confetti** — Celebra con confetti cuando completas una meta al 100%
- **100% offline** — Los datos se guardan en tu dispositivo con localStorage
- **Diseño iOS nativo** — Tema oscuro con glassmorfismo, animaciones fluidas y tipografía SF Pro

## Instalación como app en iPhone

1. Abre la URL en **Safari**
2. Toca el botón de **Compartir** (cuadrado con flecha)
3. Selecciona **Agregar a pantalla de inicio**
4. Listo — se abre como app nativa sin barra del navegador

## Stack

- HTML / CSS / JavaScript puro (un solo archivo)
- Sin frameworks ni dependencias
- Desplegado en [Vercel](https://vercel.com)

## Seguridad

- Headers de seguridad configurados en `vercel.json` (CSP, X-Frame-Options, etc.)
- Los datos se almacenan únicamente en el navegador del usuario (localStorage)
- No hay backend ni base de datos
